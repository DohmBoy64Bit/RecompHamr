# MCP Server Setup

Each built-in MCP server has its own setup guide. Choose your server below, or
see the [common reference](mcp-common.md) for shared env vars and management
commands.

| Server | Setup guide | Default tools |
|---|---|---|
| `ghidra` | [mcp-ghidra.md](mcp-ghidra.md) | 20 of 100+ RE tools |
| `n64-debug-mcp` | [mcp-n64-debug-mcp.md](mcp-n64-debug-mcp.md) | 47 debugging tools |
| `pcrecomp` | [mcp-pcrecomp.md](mcp-pcrecomp.md) | 8 of 30+ pipeline tools |
| `mcp-pine` | [mcp-pine.md](mcp-pine.md) | RPCS3 debug tools |
| `objdiff` | [mcp-objdiff.md](mcp-objdiff.md) | Diff + report tools |
| `pcsx2` | [mcp-pcsx2.md](mcp-pcsx2.md) | 30 PS2 debug tools |
| `bizhawk` | [mcp-bizhawk.md](mcp-bizhawk.md) | 16 debug tools |
| `sega2asm` | [mcp-sega2asm.md](mcp-sega2asm.md) | Split + detect tools |

For MCP architecture and how tools reach the LLM, see **[mcp.md](mcp.md)**.
