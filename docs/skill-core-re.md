# core-re

General reverse-engineering workflow skill. Load it for any RE task.

## What it teaches

- Inspect the repository before acting — identify source-of-truth files first
- Capture evidence before changing behavior
- Make the smallest useful edit, verify narrowly, iterate
- Update `.rehamr/CHANGELOG.md` or `.rehamr/EVIDENCE.md` when running `/init-re`
- Never guess binary behavior — inspect or mark as HYPOTHESIS

## When to use

Any unfamiliar codebase, binary analysis, or recompilation project. This is
the default starting skill — it establishes the evidence-first workflow.
