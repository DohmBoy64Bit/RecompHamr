# objdiff Setup

Object file diffing and match validation via encounter/objdiff. Compares
compiled object files against original assembly targets for matching
decompilation projects.

## Dependencies

- **objdiff-cli** — installed from encounter/objdiff
- **objdiff.json** — project config listing all units with target/base paths
- **Ninja** or Make — the project build system

## Install

1. Install objdiff-cli:
   `cargo install --locked --git https://github.com/encounter/objdiff.git objdiff-cli`
2. Ensure `objdiff-mcp` is on PATH (or set `RECOMPHAMR_MCP_OBJDIFF_COMMAND`)
3. Project must have `objdiff.json` config (generated by `configure.py` or manually)

## Enable

1. Start recomphamr — connect with `/mcp connect objdiff`
2. Or use HTTP: set `RECOMPHAMR_MCP_OBJDIFF_URL` to a streamable-http endpoint
3. Run `/skill objdiff` — unlocks `objdiff.*` tools
4. Verify: `/mcp tools objdiff`

Refer to [common.md](mcp-common.md) for shared env vars and management commands.
