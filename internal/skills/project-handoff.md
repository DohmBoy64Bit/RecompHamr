# project-handoff

Use this skill when preparing context for another AI, another developer, or a future session.

Create a compact handoff with:
- Project goal.
- Current confirmed state.
- Exact repo/toolchain assumptions.
- Recent commands and results.
- Files changed.
- Evidence files updated.
- Open blockers.
- Next 3 concrete steps.

Avoid long speculation. Put uncertain items under HYPOTHESIS or TODO.
